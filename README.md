## omitsu — フルスタックエンジニア

Java / Spring Boot をベースとしたバックエンドエンジニア。直近では React / TypeScript でのAPI開発を経験し、独立後に Next.js 16 で技術メディア [32blog.com](https://32blog.com) をゼロから設計・構築。フロントからインフラまで一人で完結できます。

日常的に **Claude Code（CLI）** をメイン開発ツールとして使用し、AIが書いたコードの品質をシステムで担保する仕組みを構築・運用しています。

### Tech Stack

**Backend:** Java, Spring Boot, Node.js, REST API
**Frontend:** React, Next.js (App Router / RSC), TypeScript, Tailwind CSS
**Database:** PostgreSQL, Prisma ORM
**Infra:** Vercel, Docker, Linux (WSL2), GitHub Actions
**AI Tools:** Claude Code (CLAUDE.md / Hooks / サブエージェントで品質管理)
